# app-album 加密相册

## 定位

面向隐私敏感用户的加密私密相册应用，提供密码保护、加密存储、Face ID 解锁和闯入者检测等安全能力，确保用户的私密照片、视频和文件在设备上得到端到端保护。

## 核心目标

1. **安全存储**：导入的媒体/文件加密存储在 app 私有目录，不依赖外部 URI。
2. **访问控制**：密码解锁 + Face ID 双因子认证，闯入者拍摄记录。
3. **完整管理**：目录分类、回收站、批量操作、搜索、收藏、排序。
4. **安全备份**：加密备份/恢复、安全导出传输。

## 目标用户

- 隐私敏感的普通手机用户
- 需要保护个人照片/视频不被他人查看的用户
- 对设备上敏感文件有加密存储需求的用户

## MVP 范围

### P0 必须闭环

1. 真实私密文件仓库：导入时复制到 app 私有目录，删除时清理文件
2. 加密存储与密码安全：SHA-256+salt 密码哈希、AES-256-CBC 文件加密、旧格式迁移
3. Face ID 接入解锁流程：真实认证、失败回退密码

### P1 功能闭环

4. 目录创建和重命名输入修复
5. 目录删除进入回收站，支持恢复
6. 导入来源按类型闭环（移除假入口）
7. 内容详情能力增强（视频播放、导出、移动目录）

### P2 产品完善

8. 闯入者拍摄闭环（保存照片+记录）
9. 加密备份与恢复
10. 手机互传降级为安全导出/导入
11. 自动锁定增强（前台闲置、返回锁定前页面）
12. 基础相册管理（批量、搜索、排序、收藏）

## Workspace 集成预期

- 通过 `app-center` 启动和管理
- `app-album` 是隐藏 app，不声明桌面入口 skill
- 依赖 `app-center` 的 AppBridge 进行返回导航
- 独立可构建、可安装，不依赖其他子 app 的代码
